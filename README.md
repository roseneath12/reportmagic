The **reportmagic** package provides small, expressive helper functions
that make it easier to write clear, human‑readable analytical reports.
It focuses on three core tasks that appear in almost every publication
workflow:

- extracting a single numeric value from a dataset,
- describing numbers in natural language,
- formatting numbers cleanly for publication.

These helpers are designed to be simple, predictable, and easy to
combine inside larger reporting pipelines.

## Installation

You can install the development version from GitHub:

``` r
# install.packages("remotes")
remotes::install_github("roseneath12/reportmagic")
```
